## Deployment

### Vercel (Production)
The site is deployed on Vercel at [https://onexmedia.vercel.app/](https://onexmedia.vercel.app/)

The Vite configuration uses `base: "/"` for Vercel deployment at the root path.

### Local Development
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
npm run preview
```
